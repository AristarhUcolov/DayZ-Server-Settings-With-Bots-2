## ๐ Quick Start

### Requirements

- DayZ Dedicated Server
- All mods from the [Mod Collection](https://steamcommunity.com/sharedfiles/filedetails/?id=2966726621)
- Windows OS

### Installation

1. **Download** the repository files
2. **Install** all required mods from Steam Workshop
3. **Edit** `start.bat` - Update paths to match your server location
4. **Copy** configuration files to your DayZ server directory
5. **Extract** `profiles.zip` and `keys.zip`
6. **Run** `start.bat` to launch your server
